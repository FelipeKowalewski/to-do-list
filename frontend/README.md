## Descrição
Interface web para o gerenciamento de tarefas:
- Listagem, criação e edição de tarefas
- Estilização simples e responsiva com Tailwind CSS
- Placeholders, hover e validações básicas

## Ferramentas

- Vite --> Build e dev server
- React --> Interface
- Typescript --> Linguagem
- Tailwind CSS --> Estilização
- Axios --> Comunicação com backend
- React Hooks --> Gerenciamento de estado e ciclo de vida
- Node.js --> Ambiente

## Instalação

```bash
# instala todas dependências
npm install
# ou
yarn install

# executa aplicação
npm run dev

# Servidor disponível em: http://localhost:5173
